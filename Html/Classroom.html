CNS – Computer Networks and Security
Study of communication between computers and securing that communication using cryptographic techniques, protocols, and security models.
CG – Computer Graphics
Focuses on the creation, manipulation, and representation of visual images using computers. Includes 2D/3D graphics, algorithms, and rendering techniques.
WP – Web Programming
Covers web technologies including HTML, CSS, JavaScript, and server-side scripting. Helps build dynamic, responsive websites and web applications.
SS – System Software
Explores the design and development of system-level software like compilers, assemblers, linkers, loaders, and operating system modules.
WPLB – Web Programming Lab
Practical session for building websites using HTML, CSS, JavaScript, and server-side scripts. Helps develop full-stack understanding.
CGLB – Computer Graphics Lab
Hands-on experiments using C/C++ graphics libraries to implement drawing algorithms, transformations, clipping, and animation basics.
MPLB – Mini Project Lab
Encourages students to develop a small-scale software project applying the concepts learned throughout the semester in a team.
